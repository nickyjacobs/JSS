# Security Policy

## Reporting a Vulnerability

If you find a security issue in this project:

- Do **not** open a public GitHub Issue.
- Contact the maintainers privately (e.g. via GitHub or email listed in the repo).
- Include a short description and steps to reproduce.

We will respond and work on a fix or mitigation where appropriate.

## Responsible Use

This suite includes network scanners, vulnerability scanners, and intrusion detection tools. Use them only:

- On systems and networks you are authorised to test.
- In compliance with applicable laws and policies.

Unauthorised scanning or monitoring of third-party systems is not permitted.
