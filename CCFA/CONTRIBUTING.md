# Contributing to Caesar Cipher Decoder

Bedankt voor je interesse in het bijdragen aan dit project! 

## Development Setup

1. Clone de repository
2. Zorg dat je Python 3.6+ hebt geïnstalleerd
3. Run de tests om te verifiëren dat alles werkt:
   ```bash
   python3 test_decoder.py
   ```

## Code Style

- Volg PEP 8 Python style guide
- Gebruik type hints waar mogelijk
- Voeg docstrings toe aan functies en classes
- Houd code simpel en leesbaar

## Testing

Voeg tests toe voor nieuwe features:
- Test edge cases
- Test met verschillende tekst lengtes
- Test zowel encryptie als decryptie

## Pull Requests

1. Fork de repository
2. Maak een feature branch (`git checkout -b feature/amazing-feature`)
3. Commit je changes (`git commit -m 'Add amazing feature'`)
4. Push naar de branch (`git push origin feature/amazing-feature`)
5. Open een Pull Request

## Feature Requests

Voel je vrij om issues aan te maken voor nieuwe features of verbeteringen!

## Bug Reports

Bij het melden van bugs, graag:
- Beschrijf het probleem duidelijk
- Voeg voorbeelden toe van input/output
- Vermeld je Python versie en OS
